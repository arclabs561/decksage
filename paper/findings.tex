\section{Root Cause Analysis}

\subsection{Wrong Objective Function}
Current systems optimize:
$$\min_{c'} \text{Distance}(c_{query}, c')$$

But deck improvement requires:
$$\max_{swap} Q(D_{after}) - Q(D_{before})$$

Similarity $\neq$ Improvement.

\subsection{Missing Heterogeneous Structure}
Research shows multi-modal (features + meta + image) achieves 68\%.
Our homogeneous graph collapses:
\begin{itemize}
\item Pack co-occurrence (noise)
\item Deck choices (signal)
\item Tournament results (quality)
\item Format/archetype context
\end{itemize}

Into single edge type, losing information.

\subsection{Export Pipeline Bug}
Metadata exists (format, archetype, placement) in extracted files but export command collapses to:
\begin{verbatim}
NAME_1,NAME_2,COUNT_SET,COUNT_MULTISET
\end{verbatim}

Losing deck context entirely.
